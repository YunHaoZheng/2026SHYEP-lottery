# **🏮 2026 三和尾牙摸彩活動 (2026 Sanhe Lucky Draw)**

這是一個專為手機與平板瀏覽設計的互動式尾牙摸彩網頁應用程式。採用 React \+ Tailwind CSS 開發，具備響應式設計、倒數計時鎖定機制以及確定性隨機抽獎算法。

## **✨ 特色功能**

* **⏱️ 倒數計時鎖定**：  
  * 活動設定於 **2026年1月23日 19:15** 正式開放。  
  * 在時間到達前，系統會鎖定抽獎功能，並顯示倒數計時與提示訊息。  
* **🔒 公平公正的確定性抽獎**：  
  * 採用 Seeded Random Number Generator (Mulberry32 算法)。  
  * 在 **19:14:30** (活動前30秒) 系統會自動鎖定全體的配對結果。  
  * 確保所有人在同一時間點按下抽獎，獲得的結果與總表完全一致，無需後端資料庫即可達成同步效果。  
* **📱 響應式互動介面**：  
  * 針對手機直式操作優化。  
  * 包含試玩模式、圖片放大檢視、總表查詢等功能。  
  * 充滿年節氣氛的紅金配色與動畫效果。

## **🛠️ 技術架構**

本專案採用 **Standalone HTML** 架構，無需 Node.js 環境或 Build Process 即可直接部署。

* **React 18**: 透過 CDN 引入，處理前端互動邏輯。  
* **Tailwind CSS**: 透過 CDN 引入，處理 UI 樣式與響應式佈局。  
* **Babel**: 瀏覽器端即時編譯 JSX 語法。  
* **Lucide React**: 向量圖示庫。

## **🚀 如何部署**

本專案由靜態檔案組成，非常適合部署於 **GitHub Pages**。

1. 將 index.html 與所有圖片素材上傳至 GitHub Repository。  
2. 在 Repository 的 **Settings \> Pages** 中啟用 GitHub Pages。  
3. 選擇 main 分支並儲存，即可獲得公開網址。

## **📂 檔案結構**

* index.html: 包含所有程式邏輯、樣式與結構的主檔案。  
* \*.jpeg: 獎項圖片素材 (需與 html 位於同一層目錄)。

## **⚠️ 注意事項**

* 本程式碼中的圖片路徑為相對路徑，請確保圖片檔名與程式碼中的 PRIZE\_IMAGES 列表完全一致。  
* 若在本地端開啟，請直接雙擊 index.html 即可執行。
